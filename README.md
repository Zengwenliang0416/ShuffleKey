# ShuffleKey

一个简洁优雅的 iOS 数字键盘随机打乱工具。

## ✨ 功能特点

- 🎲 一键随机打乱数字键盘布局
- 🎨 精心设计的红色渐变主题
- 📱 完整支持 iPhone 和 iPad
- 🌈 优雅的动画效果
- 🎯 简单直观的用户界面

## 🛠 技术栈

- SwiftUI
- Swift 5.9
- iOS 17.0+
- Xcode 15.0+

## 🚀 开始使用

1. 克隆项目
```bash
git clone https://github.com/Zengwenliang0416/ShuffleKey.git
```

2. 打开项目
```bash
cd ShuffleKey
open ShuffleKey.xcodeproj
```

3. 运行项目
- 选择目标设备（iPhone 或 iPad）
- 点击运行按钮或按下 `Cmd + R`

## 🔧 开发环境要求

- macOS Sonoma 14.0+
- Xcode 15.0+
- iOS 17.0+
